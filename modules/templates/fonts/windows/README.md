# Local Windows fonts

Put copied Windows or Microsoft Office font files here. The `programs.customFonts`
module packages every `.ttf`, `.ttc`, and `.otf` file in this directory and adds
them to fontconfig through Home Manager.

Recommended WPS formula fonts:

- `symbol.ttf`
- `webdings.ttf`
- `wingding.ttf`
- `WINGDNG2.TTF`
- `WINGDNG3.TTF`
- `MTEXTRA.TTF`

Recommended Chinese document compatibility fonts:

- `simsun.ttc`
- `simhei.ttf`
- `msyh.ttc`
- `msyhbd.ttc`
- `simfang.ttf`
- `simkai.ttf`

Optional Microsoft Office compatibility fonts:

- `calibri*.ttf`
- `cambria*.ttf`
- `consola*.ttf`
